# Geometric Library - Документация

## Общее описание решения

Библиотека geometric_lib предоставляет функции для вычисления площади и периметра геометрических фигур.

## Описание функций

### Прямоугольник (rectangle.py)

#### area(a, b)
Вычисляет площадь прямоугольника.

Параметры:
- a (float): длина
- b (float): ширина

Возвращает:
- float: площадь прямоугольника

### Круг (circle.py)
#### area(r)
Вычисляет площадь круга.

Параметры:

r (float): радиус

Возвращает:

float: площадь круга

perimeter(r)
Вычисляет длину окружности.

Параметры:

r (float): радиус

Возвращает:

float: длина окружности

### Квадрат (square.py)
#### area(a)
Вычисляет площадь квадрата.

Параметры:

a (float): длина стороны

Возвращает:

float: площадь квадрата

perimeter(a)
Вычисляет периметр квадрата.

Параметры:

a (float): длина стороны

Возвращает:

float: периметр квадрата

### Треугольник (triangle.py)
#### area(a, h)
Вычисляет площадь треугольника.

Параметры:

a (float): основание

h (float): высота

Возвращает:

float: площадь треугольника

perimeter(a, b, c)
Вычисляет периметр треугольника.

Параметры:

a (float): первая сторона

b (float): вторая сторона

c (float): третья сторона

Возвращает:

float: периметр треугольника
